# Hooks

Hooks should be grounded in a real thesis, artifact, or tension. Do not write fake drama.

## Families

### Contradiction

Use when the post pushes against a common belief.

```txt
AI agents don't need better plans. They need better review.
Local-first is wrong for public feeds, and that is the point.
Your group chat is already a social network.
```

### Confession

Use when the author has a real behavior change, doubt, or mistake.

```txt
I stopped reading my agents' plans, and that scared me.
I used to think markdown was enough for agent output. I don't anymore.
I keep approving work I have not really verified.
```

### Specific Problem

Use when the pain is concrete.

```txt
Your coding agent is producing more work than you can verify.
Most agent plans are too long to review.
Screenshots beat status reports because they remove interpretation.
```

### Before And After

Use when there is a workflow change.

```txt
I used to ask agents for markdown. Now I ask for HTML.
I used to write launch posts by hand. Now I write one source article and compile the rest.
I used to treat comments as engagement. Now I treat them as source material.
```

### Uncomfortable Truth

Use when the post should invite disagreement.

```txt
The human in the loop is usually just approving paperwork.
Most AI coding demos skip the review step.
The bottleneck did not disappear. It moved.
```

### Artifact First

Use when the visual is the point.

```txt
This diagram changed how I think about agent review.
This diff explains the product decision better than the roadmap.
This screenshot is the whole problem.
```

## Platform Notes

LinkedIn hooks can be a little more complete:

```txt
AI agents don't need longer plans. They need outputs humans can actually verify.
```

X hooks should be tighter:

```txt
The bottleneck is verification.
```

TikTok, Reels, and Shorts need immediate tension:

```txt
Your AI coding workflow has a review problem.
```

Instagram can be slightly more personal:

```txt
What changed how I review AI-generated code.
```

Reddit hooks should sound like a real post title:

```txt
I changed my agent workflow from markdown plans to HTML reports. Here is what got better.
```

## Anti-Patterns

- "The future of..."
- "Everything you need to know about..."
- "This changes everything"
- "Nobody is talking about..."
- "I asked AI to..."
- Fake personal stakes
- Claims without a concrete artifact
