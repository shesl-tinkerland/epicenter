# README Refresh

The READMEs haven't kept up with the project's evolution. The core thesis (local-first, open-source apps sharing a common memory) is the same, but the business model has crystallized (open-core, split licensing, Epicenter Cloud) and the developer platform story is now real. The root README still leads with a migration banner from months ago and doesn't mention the workspace library at all.
